# Plan: 04-01 World Builder Core

Implement the extraction of policy parameters and synthesis of an agent population utilizing structured LLM parsing.

## User Review Required

> [!IMPORTANT]
> **World Builder Strategy**:
> - We will create a `WorldBuilder` class that handles document ingestion and agent generation.
> - Default agent count is 40 (with 3-5 key figures), which aligns with the original vision.
> - The output of the generation will be returned but not immutably locked, allowing for a "God Mode: Setup" step in the UI later where agents can be edited before starting the simulation loop.
