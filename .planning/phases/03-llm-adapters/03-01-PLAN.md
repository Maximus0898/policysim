# Plan: 03-01 LLM Adapters

Implement a unified reasoning layer with support for Anthropic, OpenAI, Ollama, and LM Studio.

## User Review Required

> [!IMPORTANT]
> **Provider Strategy**:
> - We will use the official `anthropic` and `openai` Python SDKs.
> - **Ollama** and **LM Studio** will share the `OpenAIProvider` class via custom `base_url` settings.
> - **Tenacity** will be used for exponential backoff on retries.
