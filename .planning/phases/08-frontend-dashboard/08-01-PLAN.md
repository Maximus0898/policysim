## User Review Required

> - Hooks up the new Vite configuration and sets up the root `app.html` with a modern sans-serif tech font (Inter/Outfit) to fulfill the premium aesthetics directive.

Scaffold the SvelteKit project template.

<task>

4. Install standard layout styling in `frontend/src/app.css` using typical `@import "tailwindcss"` syntax for Tailwind v4.
